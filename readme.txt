== Description ==

Koo Publisher is an official plugin maintained by KooApp team.
It allows the users to auto-post articles on the associated Koo App account. User has the control to toggle autopost at an article level. Additionally, this plugin also help wordpress based sites to embed Koos by pasting the URL of the Koo.


This plugin adds the following:

* Admin can setup the autoposting by adding the Koo API key in the settings screen of the plugin. 
* Option to enable/ disable the plugin.
* Auto post feature can be toggled from the post editor screen
* Enables Koo post embed feature using the URL of the Koo
